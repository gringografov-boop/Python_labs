# Grafov Library Simulation

Лабораторная работа №4: симуляция работы городской библиотеки с индексированием коллекции книг по ISBN, автору и году издания.

## Структура проекта

- `src/models/` — классы книг (`Book`, `PaperBook`, `EBook`).
- `src/index_dict.py` — индексированная коллекция (`IndexDict`) по ISBN/автору/году.
- `src/library.py` — класс `Library`, работа с коллекцией и статистикой.
- `src/simulation.py` — сценарий симуляции (случайные операции с книгами).
- `main.py` — точка входа, запускает симуляцию.

## Зависимости

Проект использует только стандартную библиотеку Python ( `random`, `dataclasses`, `typing` и т.п.), внешних пакетов нет. [web:259][web:265]  

Требуется:

- Python 3.10+.

## Описание симуляции

При запуске выводится:

- стартовое количество книг;
- лог из N шагов (добавление, удаление, поиск, проверка индексов);
- финальная статистика: количество книг, уникальные авторы, годы, жанры.

Сид и число шагов можно изменить в `main.py` при вызове `run_simulation(...)`.
